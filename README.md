# 🚖 Gojek Service Sentiment Analysis

## 📌 Overview
This project analyzes public sentiment toward Gojek services using Twitter (X) data. The objective was to identify customer satisfaction patterns and evaluate machine learning models for sentiment classification.

## 🛠️ Tools & Technologies
- Python
- RapidMiner
- Excel
- Twitter API

## ⚙️ Methodology
- Data collection using Twitter API
- Data preprocessing (cleaning, tokenization, stopword removal, stemming)
- Manual sentiment labeling
- Model comparison using:
  - Naïve Bayes
  - Support Vector Machine (SVM)
  - Random Forest

## 📊 Key Results
- Best Model: Naïve Bayes
- Accuracy: 75.09%
- F1-Score: 74.63%

Main negative sentiment factors identified:
- Technical application errors
- Customer service responsiveness
- Driver cancellation issues

## 💡 Business Insights
The analysis suggests that improving system stability and customer support responsiveness could help enhance user satisfaction and reduce negative sentiment toward Gojek services.
